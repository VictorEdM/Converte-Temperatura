<h1>Conversor de Temperaturas</h1>

<p>Objetivo</p>
 <ul>
  <li>Converter uma temperatura X em uma temperatura Y (Celsius, Kelvin e Fahrenheit).</li>
 </ul>

<p>Ferramentas</p>
 <ul>
  <li>PyQt5</li>
  <li>Qt Designer</li>
 </ul>

<p>Linguagem</p>
 <ul>
  <li>Python (3.8)</li>
 </ul>
 
<p>Instalar requerimentos</p>
 <ul>
  <li>pip install -r requirements.txt</li>
 </ul>
